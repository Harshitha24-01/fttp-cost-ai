# 🚀 FTTP Cost Intelligence Platform

An AI-powered telecom network planning system that automates FTTP (Fiber to the Premises) cost estimation, route optimization, and business analysis using Agentic AI.

---

## 📌 Overview

This project is designed to help telecom providers efficiently plan fiber network deployment by:

- Automating cost estimation  
- Generating multiple network routes  
- Analyzing profit and ROI  
- Providing AI-driven recommendations  
- Visualizing routes on an interactive map  

---

## 🎯 Key Features

### 📊 Cost & Profit Analysis
- Calculates deployment cost based on distance, equipment, and labor  
- Computes revenue, profit, and profit margin  
- Supports realistic telecom financial modeling (CAPEX & OPEX)

---

### 🗺️ Multi-Route Planning
- Generates multiple routes between source and destination  
- Classifies routes:
  - Shortest Path  
  - Low Cost Path  
  - Risky Path  
  - Eco-Friendly Path  

---

### 🤖 AI-Powered Insights (LLM)
- Explains cost and deployment strategy  
- Suggests optimization techniques  
- Provides beginner-friendly guidance  

---

### 💰 Budget Optimization
- Tracks budget utilization  
- Suggests cost-saving strategies  
- Supports deployment type selection:
  - Underground  
  - Aerial  
  - Hybrid  

---

### 📍 Interactive Map
- Visualizes routes using map  
- Displays multiple paths with color coding  
- Shows route details (distance, cost, risk)

---

### 🧑‍💻 User-Friendly UI
- Dashboard with analytics  
- Tooltips for each input field  
- Beginner tutorial & help panel  

---

↓
Backend (FastAPI)
↓
Cost Engine + Route Optimizer + AI (LLM)
↓
Map Services (Geolocation + Routing)

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- React Leaflet (Maps)
- Chart.js / Recharts

### Backend
- FastAPI (Python)
- Geopy (distance calculation)

### AI / LLM
- Groq API (LLM-based insights)

### Others
- Axios (API calls)
- React Query (data fetching)
